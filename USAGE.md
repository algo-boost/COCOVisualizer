# 使用示例

## 快速开始

### 1. 启动应用

```bash
cd /root/cv-scripts/coco-visualizer
python3 app.py
```

或者使用启动脚本：

```bash
./run.sh
```

### 2. 访问界面

在浏览器中打开：`http://localhost:6009`

## 使用场景示例

### 场景1：上传本地COCO文件

1. 准备一个COCO格式的JSON文件（例如：`annotations.json`）
2. 在网页上，将文件拖拽到上传区域
3. 文件上传成功后，路径会自动填入"服务器路径"输入框
4. （可选）如果图片路径是相对路径，设置图片目录
5. 输入数据集名称（例如：`my_dataset`）
6. 点击"加载数据集"
7. 等待加载完成，查看数据集信息

### 场景2：使用服务器上的文件

1. 在"服务器路径"输入框中输入路径，例如：`/root/datasets/coco/annotations.json`
2. 点击"浏览"按钮浏览服务器目录
3. 在文件浏览器中找到并点击COCO JSON文件
4. 设置图片目录（如果需要）
5. 输入数据集名称
6. 点击"加载数据集"

### 场景3：生成中心点分布图（等距坐标轴）

1. 加载数据集后，在"可视化选项"中选择"中心点分布（散点图）"
2. 确保"等距坐标轴"选项已勾选
3. （可选）在类别筛选中选择要显示的类别
4. 点击"生成可视化"
5. 查看生成的散点图，x轴和y轴将保持等距

### 场景4：筛选特定类别进行可视化

1. 加载数据集后，在"筛选选项"中
2. 取消"全选"，然后只勾选感兴趣的类别（例如：只选择"cat"和"dog"）
3. 选择任意可视化类型（例如："类别数量"）
4. 点击"生成可视化"
5. 图表将只显示选中类别的数据

### 场景5：对比不同类别的bbox分布

1. 加载数据集
2. 选择可视化类型："各类别bbox分布"
3. 勾选"等距坐标轴"以保持空间比例
4. 选择要对比的类别（可以多选）
5. 生成可视化
6. 每个类别会显示在独立的子图中，便于对比

## 可视化类型说明

### 类别分布（饼图）
显示各类别在数据集中的占比情况

### 类别数量（柱状图）
显示各类别的样本数量，便于对比

### 面积分布（sqrt(area)）
显示各类别标注框面积的平方根分布，有助于了解目标大小

### 长边分布
显示各类别标注框最长边的分布

### 宽高比统计
显示各类别宽高比的均值和标准差

### 长短边比分布
显示各类别长短边比例的分布

### 宽度和高度分布
并排显示宽度和高度的分布情况

### 中心点分布（散点图）
显示所有标注的中心点在图像中的分布位置
- **推荐启用等距坐标轴**，以保持空间比例

### 各类别bbox分布
为每个类别单独显示bbox的空间分布
- **推荐启用等距坐标轴**，以保持空间比例
- 颜色表示宽度，大小表示高度

## 技巧和提示

1. **等距坐标轴**：对于散点图类型的可视化，启用等距坐标轴可以更准确地反映空间关系

2. **类别筛选**：使用类别筛选可以：
   - 聚焦于感兴趣的类别
   - 减少图表复杂度
   - 提高对比分析效率

3. **大数据集**：如果数据集很大，加载和生成可视化可能需要一些时间，请耐心等待

4. **图片目录**：如果COCO JSON中的图片路径是相对路径，必须设置正确的图片目录，否则某些功能（如颜色分析）可能无法使用

5. **多次可视化**：可以多次生成不同类型的可视化，无需重新加载数据集

## 常见问题

**Q: 上传文件后找不到？**
A: 上传的文件保存在 `uploads/` 目录中，可以通过服务器路径访问。

**Q: 为什么等距坐标轴选项对某些图表没有效果？**
A: 等距坐标轴主要适用于散点图类型的可视化（如中心点分布、bbox分布）。对于柱状图、箱线图等图表类型，等距坐标轴不适用。

**Q: 如何清除已加载的数据集？**
A: 刷新页面即可清除当前会话中的数据。

**Q: 可以同时加载多个数据集吗？**
A: 当前版本支持在同一个会话中加载多个数据集，每个数据集有独立的ID。

**Q: 图表可以下载吗？**
A: 当前版本在浏览器中显示图表，可以右键保存图片。
