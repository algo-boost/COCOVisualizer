# 图片查看器使用说明

## 功能概述

图片查看器允许您：
1. **按类别筛选图片**：选择感兴趣的类别，只查看包含这些类别的图片
2. **左右切换**：使用按钮或键盘箭头键切换图片
3. **放大查看**：支持图片放大、缩小和重置
4. **标注显示**：在图片上显示标注框和类别标签
5. **标注信息**：查看每张图片的详细标注信息

## 使用步骤

### 1. 加载数据集
首先需要加载COCO数据集（参考主README）

### 2. 切换到图片查看标签页
点击顶部的"图片查看"标签页

### 3. 选择类别
- 在"筛选类别（多选）"区域勾选要查看的类别
- 可以选择多个类别，系统会显示包含任意选中类别的图片

### 4. 加载图片
- 点击"加载图片"按钮
- 系统会筛选出符合条件的图片列表

### 5. 查看图片
- **切换图片**：
  - 点击"← 上一张"或"下一张 →"按钮
  - 或使用键盘左右箭头键（← →）
  
- **放大缩小**：
  - 点击"放大"按钮放大图片
  - 点击"缩小"按钮缩小图片
  - 点击"重置"按钮恢复原始大小
  - 支持0.5x到5x的缩放范围

- **标注显示**：
  - 默认显示标注框和类别标签
  - 点击"显示/隐藏标注"按钮切换标注显示
  - 不同类别使用不同颜色的标注框

### 6. 查看标注信息
在图片下方会显示当前图片的所有标注信息，包括：
- 类别名称
- 标注框位置坐标
- 标注面积（如果有）

## 快捷键

- **←** (左箭头)：上一张图片
- **→** (右箭头)：下一张图片

## 标注框颜色

系统使用以下颜色区分不同类别：
- 红色 (#FF0000)
- 绿色 (#00FF00)
- 蓝色 (#0000FF)
- 黄色 (#FFFF00)
- 紫色 (#FF00FF)
- 青色 (#00FFFF)

如果类别超过6个，颜色会循环使用。

## 注意事项

1. **图片路径**：确保在加载数据集时正确设置了图片目录
2. **标注框位置**：标注框基于COCO格式的bbox坐标绘制
3. **缩放性能**：放大图片时，标注框会相应调整大小
4. **图片加载**：首次加载图片列表可能需要一些时间，请耐心等待

## 技术细节

- 图片通过后端API `/api/get_image` 获取
- 标注框使用HTML5 Canvas绘制
- 支持实时缩放和标注重绘
- 标注框位置根据图片显示尺寸自动缩放
